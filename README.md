# Security Event Explorer (React + Python + Postgres)

A full-stack web application for browsing, filtering, and annotating security alerts generated by my home SOC lab (Wazuh, Suricata).

This project turns raw JSON alerts into an interactive UI so analysts can quickly review events, explore details, and add notes â€” similar to a lightweight SIEM front end.

It complements my other SOC automation projects by providing a **visual interface** on top of the existing alert pipeline and detections.

---

## ğŸ” What This App Does

- **Displays alerts in a searchable dashboard**
  - Alerts from Wazuh/Suricata ingested into a Postgres database.
  - React UI shows timestamp, severity, source, rule, IPs, tags.

- **Filtering & search**
  - Filter by severity, source, text search, date range.

- **Detailed alert views**
  - Expand a single alert to see full JSON, metadata, tags, and context.

- **Analyst notes**
  - Add/edit notes tied to specific alerts.
  - Stored in the backend and linked to user accounts.

- **Authentication**
  - Simple JWT-based login for analysts.

- **API-powered**
  - Python FastAPI backend exposes `/alerts`, `/alerts/{id}`, `/notes`, etc.

- **Optional cloud deployment**
  - Dockerized backend & frontend.
  - Deployment-ready for AWS (EC2/ECS + RDS + S3/CloudFront).
